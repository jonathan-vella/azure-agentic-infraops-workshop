# Sample Team Folder

> **This is a template** showing the expected folder structure and file formats.
> Teams should NOT copy this content — generate your own using the agents!

This folder demonstrates what a completed hackathon submission looks like.

## Expected Structure

```
agent-output/{team}/
├── 01-requirements.md           ← From @plan
├── 02-architecture-assessment.md ← From @architect  
├── 03-des-architecture-diagram.md (optional)
├── 04-implementation-plan.md    ← From @bicep-plan
├── 05-load-test-results.md      ← Challenge 5
├── 06-deployment-summary.md     ← From @deploy
└── score-results.json           ← Generated by scoring script

infra/bicep/{team}/
├── main.bicep                   ← Entry point
├── main.bicepparam              ← Parameters
├── deploy.ps1                   ← Deployment script
└── modules/
    ├── app-service.bicep
    ├── sql-database.bicep
    ├── storage.bicep
    ├── key-vault.bicep
    └── monitoring.bicep
```
